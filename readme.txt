---------------------------------------------------------
NK BLUEPRINT READ ME.txt
---------------------------------------------------------

NK Blueprint is a community project, driven by users on the NimbleKit forum:
http://www.nimblekit.com/forum/


Aims:
- To lower the learning curve for new users
- To increase community involvement to take stress off the core NK developers
- To illustrate and extend the NK documentation by providing working commented code with description of the codes usage

Objectives:
- To release and maintain the NKBlueprint app which is a tech demo of every available NK feature
- To release and maintain smaller apps forked from NK Blueprint that use NK to solve specific use case scenarios
